# 関連チケット横断検索・分析

指定したテーマに関連するチケットを多角的に検索し、分析結果を提供します。

## 実行手順

1. **テーマ確認**: ユーザーが指定したテーマ（例：「認証機能」「データベース障害」「A社対応」）を確認
2. **キーワード生成**: テーマから関連する検索キーワードを5-8個自動生成
3. **横断検索**: 各キーワードでチケットを検索し結果を収集
4. **結果分析**: 以下の観点で分析・整理
   - チケット数の多いキーワード
   - 頻出するプロジェクト・担当者
   - ステータス分布（未解決/解決済み）
   - 時系列での傾向
5. **サマリー提供**: 
   - 関連チケット総数
   - 主要な課題・パターン
   - 注目すべきチケット（未解決の重要案件等）
   - 次のアクション提案

## 出力形式
- キーワード別検索結果一覧
- 統計サマリー（件数、プロジェクト分布等）
- 重要チケットのハイライト
- 傾向分析と推奨アクション

検索テーマを教えてください。
