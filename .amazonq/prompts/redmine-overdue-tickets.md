# 期限切れRedmineチケット一覧取得

以下の手順で自身の期限切れチケットを取得してください：

1. まずRedmineにログインしてください
2. 利用可能なトラッカー一覧を取得してください
3. 除外したいトラッカーがあれば教えてください（例：「質問」「要望」など）
4. 以下の条件でチケットを検索してください：
   - 担当者：自分（ログインユーザー）
   - ステータス：未完了（進行中、新規など）
   - 期限：今日より前（期限切れ）
   - 除外トラッカー：指定されたトラッカーを除く

検索結果は以下の形式で表示してください：
- チケット番号、件名、期限日、プロジェクト名、トラッカー名
- 期限切れ日数の降順でソート
- 合計件数も表示

必要に応じて個別チケットの詳細も取得できます。
